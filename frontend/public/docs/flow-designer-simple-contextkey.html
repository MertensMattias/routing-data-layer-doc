üìä Simple contextKey

‚úÖ Phase 2 Implementation
contextKey is a simple string field that tags transitions with the context variable that influenced the routing decision.

üìã Use Cases

Use Case Description
Tracking Log which context influenced routing
Analytics Report on context usage patterns
Debugging Understand why a path was taken
Auditing Track context-based decisions

üîÑ How It Works

Important
The contextKey does NOT affect routing logic. It's metadata only. Both PREMIUM and STANDARD can have the same contextKey value.

üìù Example Flow

  1. Intent segment executes
  2. Returns result: PREMIUM
  3. Logs: customerType influenced decision
  4. Routes to Premium Flow
  5. Context metadata stored for analytics
